TwitterClient by gistinc

A Java client for the Twitter Streaming API

Java client to connect to the Twitter Streaming API. TwitterClient uses Jakarta Commons HttpClient 3.1, http://hc.apache.org/httpclient-3.x/. It backs off and reconnects on HTTP and TCP errors as per the spec. It can connect using multiple sets of credentials at once. Actual processing of the Twitter stream is provided a help object.

Dependencies

Jakarta Commons HttpClient 3.1
